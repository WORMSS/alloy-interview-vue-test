## Project setup

Clone and install dependencies with:

```
npm install
```

### Development

To run the dev server use:

```
npm run serve
```

### Unit tests

Don't let your unit tests slip, execute them with:

```
npm run test:unit
```

### End-to-end tests

The icing on the cake, test end-to-end:

```
npm run test:e2e
```
